Search App Store and Play Store for: $ARGUMENTS

Use WebSearch to find the relevant apps on both stores:
- `site:apps.apple.com $ARGUMENTS`
- `site:play.google.com $ARGUMENTS`

For the top 2–3 apps per store, use WebFetch on the app listing page to extract:
- Overall rating and total review count (market size signal)
- Recent reviews — sort by "Most Recent" if possible to surface current sentiment
- Critical reviews (1–2 star) — these reveal recurring frustrations competitors can solve
- Top positive reviews (5 star) — what users love and want more of
- App description and screenshots alt-text (positioning language)
- In-app purchase structure if listed
- Update frequency (active development signal)

Also note the number of competing apps returned for the search query — this is a quick proxy for category saturation.
